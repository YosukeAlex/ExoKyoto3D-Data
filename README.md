# ExoKyoto3D-Data

Data repository for ExoKyoto3D preview testing and updates.

## Overview

This repository provides data files, configuration information, and testing materials for ExoKyoto3D, a 3D visualization platform for exploring exoplanetary systems, stellar environments, and planetary habitability.

The ExoKyoto3D application binary is expected to be released separately through NASA/GSFC or related official release channels. This repository is maintained to support preview testing, data updates, and application-side data access.

## Data Source and Attribution

The data included in this repository has been compiled and reformatted for use with ExoKyoto3D. It is based on a combination of publicly available astronomical databases and literature sources, including, but not limited to, the NASA Exoplanet Archive, the Extrasolar Planets Encyclopaedia (exoplanet.eu), NASA ADS-linked publications, other published scientific data, and original ExoKyoto data archives developed by Yosuke A. Yamashiki and collaborators.

Some additional research data, annotations, classifications, derived parameters, and visualization-oriented data structures were prepared by Yosuke A. Yamashiki for ExoKyoto-related research, education, and visualization purposes.

The data may be updated, corrected, reformatted, or replaced without prior notice. The latest update date and version information will be provided separately in the version information or update section.

This repository does not claim ownership of the original public-domain, open-access, or third-party source data. Users should refer to the original data providers, databases, and publications for authoritative datasets, citation requirements, licensing conditions, and terms of use.

## Binary Data Format

The data file provided here is distributed in a binary format specifically designed for use with the ExoKyoto3D application developed by Yosuke A. Yamashiki.

This repository is intended to provide data files, updates, and testing materials that can be referenced by the ExoKyoto3D application. The binary data format may change in future versions. Compatibility with all future versions of the ExoKyoto3D application is not guaranteed unless explicitly stated.

The binary file is not intended to serve as an authoritative standalone astronomical database. It is a compiled and reformatted application data package for visualization and educational/research demonstration purposes.
